## 프로덕트 해라의 핵심가치
Enjoy, think, solve
- 스스로 결정하는 힘을 기를 수 있습니다.
- 자기주도적인 회고를 활용할 수 있으면서도, 같이 고민하는 즐거움을 느낄 수 있습니다.
